# Support

Thanks for using WinPilot. For help, please open a GitHub issue:

- https://github.com/chaitanyame/winpilot/issues

## What to include

- Windows version (10/11) and build number
- WinPilot version
- Steps to reproduce
- Expected vs. actual behavior
- Relevant logs (see `logs\debug.log`)

## Security issues

Please do **not** open public issues for security reports. See [SECURITY.md](SECURITY.md).
